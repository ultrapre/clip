VN/Eroge Script sizes

Jump to: navigation, search

This page is intended to be a collection of various scenario sizes one would get by extracting an eroge/VN's Japanese script and making a linecount/KB count on it. Feel free to add any Japanese eroge/VN you may encounter. (Don't worry about filling out every column, it's not a requirement.)
Please only add a game if you yourself, or someone trustworthy (ie. someone who knows what he's doing) counted it, rather than hearing it somewhere from a dubious source. We'd like to avoid spreading incorrect information.

If you are looking for something to count the script size with, then depending on what engine the game is on, use the Kirikiri line/byte counter, NScripter line/character/byte counter or use the Nitroplus NPS line/byte counter (doesn't work with the Nitroplus games on N2System). To count unique kanji use frequency.py or JWCPe.

Disclaimer:

Unless detailed in a reference below, the numbers below are only representing the pure, sheer text one would need to translate, without any codes, commands (that includes furigana reading/"Ruby text"/"R Text" command too, ie, the small words appearing over certain words, like name/term pronunciations), and speakertags. Moreover if a game would have identical replay scripts in separate files (eroscene replay scripts for example), then keep in mind that they aren't and shouldn't be included in these numbers.
The number of unique kanji could be considered a crude measure of writing complexity.
Additionally all the scripts are, and should be taken in SJIS encoding.

Some of these numbers may not be entirely accurate, either due to being mistakenly entered as MiB or other errors on the part of the person who counted them. It would be best to treat the numbers on this chart as a rough estimate of script length rather than an exact character count.
